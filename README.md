## 📖 What is Pithos?

**Pithos (ピトス)** は、現代の腐敗した統治構造（Leviathan）に対抗するために設計された、**サーバーレス・ガバナンスOS**です。

古代ギリシャの哲学者ディオゲネスが、巨大なインフラを拒絶して「土の器（Pithos）」に住んだように、このシステムは**「スマホのブラウザ」**という最小の器だけで機能します。
サーバーも、データベースも、権威も必要ありません。必要なのは**「論理（Python）」**だけです。

## ⚙️ Modules

このOSは、国家機能を代替する4つのモジュールで構成されています。

### 1. 📊 Core (SBCM Auditor)
**「感情」を排し、「物理法則」で行政を監査する。**
SBCM理論に基づき、予算の「歪み指数 ($D_{index}$」を即座に計算。身の丈に合わないハコモノ行政（第4象限）を、数学的に検知・拒絶（Reject）します。

### 2. 💰 Treasurer (Iron Vault)
**「人間」を介さず、「コード」で金を動かす。**
恣意的な経理操作を排除するため、全銀フォーマット（Zengin Format）の振込データをアルゴリズムが自動生成します。生成されたデータには改ざん防止のハッシュ（Digital Seal）が刻印されます。

### 3. 👁️ Sphinx (Citizen Oracle)
**「報告」を信じず、「目」を信じる。**
市民の隙間時間を活用した分散型監査システム。現場写真の真偽を不特定多数のユーザーが判定し、コンセンサス（合意）が得られた場合のみ、スマートコントラクトが実行されます。

### 4. 🏺 Ostraka (Digital Banishment)
**「権力」を恐れず、「数」で追放する。**
古代アテナイの「陶片追放」をデジタル実装。システムに対する裏切り者（ハッカー、腐敗した政治家、癒着業者）に対し、市民が投票権（Shard）を行使し、一定数を超えれば物理的にアカウントを凍結・追放します。
